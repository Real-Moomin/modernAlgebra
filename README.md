# 현대 대수학 — 디자이너를 위한 친절한 가이드

이과가 아닌 분(특히 디자인·인문 계열)을 위해 **보드게임, 일상 비유**로 현대 대수학의 핵심을 설명하는 웹사이트입니다.

## 구성

- **1단계**: 군론 (Group Theory) — 군의 정의, 예시, 부분군, 준동형, 몫군, Sylow 정리 등
- **2단계**: 환론 (Ring Theory) — 환, 가환환, 정역, 체, 아이디얼, PID/UFD
- **3단계**: 체론 (Field Theory) — 체 확장, 대수적 원소, 최소다항식, 유한체
- **4단계**: 갈루아 이론 — 갈루아 군, 대응, 다항식 가해성, 5차 방정식
- **5단계**: 모듈 이론 — 모듈, 부분모듈, 자유모듈, Noether, 구조정리
- **6단계**: 범주론 맛보기 — 범주, 함자, 자연변환, 보편성, 극한/여극한

각 챕터마다 **개념 설명 → 친숙한 비유(보드게임 등) → 연습문제 → 해설(토글)** 순서로 볼 수 있습니다.

## 사용 방법

1. `index.html`을 브라우저에서 열면 메인 목차가 나옵니다.
2. 각 단계 링크를 눌러 해당 챕터로 이동합니다.
3. 챕터 내 상단 목차로 섹션 간 이동이 가능합니다.
4. 연습문제 아래 **「해설 보기」** 버튼을 누르면 해설이 펼쳐집니다.

로컬에서 바로 열어도 되고, 원하면 아무 정적 호스팅(Netlify, GitHub Pages 등)에 올려서 사용해도 됩니다.

## 파일 구조

```

├── index.html          # 메인 목차

├── ring-theory.html   # 2단계 환론

```
